# About Markdown
---
- Italics and Bold
use underscore ( _ ) _Italics_
use double asterisks ( ** ) **Bold**
**_Bold and italics_**
use ( _** ) for both italics and bold _**Italics and bold**_

- Headers
use (#) for H1 and (######) for H6

- Links
use ([]) for the link text
use ( () ) for hyperlink
